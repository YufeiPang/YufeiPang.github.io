搭建个人博客（Hexo框架）

搭建个人博客(Hexo博客框架)

1.环境搭建

下载node.js网址：https://nodejs.org/en/（内含npm【包管理器】和node）
下载git网址：https://git-scm.com/downloads
#以管理员身份运行命令行cmd
#安装测试：cmd命令行分别键入“node -v”“npm -v”“git -v"查看版本

#全局安装镜像源指向淘宝

npm install -g cnpm --registry=https://registry.npm.taobao.org

cnpm -v

#利用cnpm安装hexo框架

cnpm install -g hexo-cli

hexo -v

2.创建博客

下载markdown文本编辑器typora网址：https://typora.io/
mkdir blog  #在指定文件夹中创建新文件夹blog

cd blog\  #进入文件夹
    

##注意，文件夹创建完毕后，后续所有代码操作均在该文件夹中进行
hexo init  #初始化博客

hexo s  #启动博客


##此时博客已在localhost:4000创建，可到浏览器中查看

2.1创建博文

hexo n "文章名"  #然后到blog文件夹下source—>_posts中查看新创建的.md文件，用typora打开并进行编辑

hexo clean #清空

hexo g #生成

hexo s #重启，然后到用浏览器搜索localhost:4000进行预览

ctrl+c #退出

3.远端部署至GitHub

（1）登录GitHub(https://github.com/)->创建new repository
（2）在命令行cmd中安装插件
cnpm install --save hexo-deployer-git #下载安装插件
（3）使用记事本打开blog文件夹中的config.yml修改配置如下：
deploy:
  type: 'git'
  repo: https://github.com/YufeiPang/YufeiPang.github.io #此处为GitHub上创建的仓库的地址
  branch: master
  
##注意，冒号后如丢失空格，会导致失败
（4）回到命令行cmd向远端部署，代码如下：
git config --global user.email "you@example.com"  #在""内输入GitHub注册邮箱地址
git config --global user.name "Your Name"  #在""内输入GitHub注册昵称

hexo d  #向远端部署
（5）再次刷新查看GitHub仓库，此后则可以通过仓库名”注册昵称.github.io“访问博客

4.更换主题

此处使用yilia主题做示范

（1）在命令行cmd中下载主题yilia
git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia  #yilia下载安装
（2）再次修改配置_config.yml
themes：yilia  ##注意，冒号后如丢失空格，会导致失败
（3）再次重复流程”清除界面——生成网页——重启预览——向远端部署“
hexo clean  #清除界面

hexo g  #生成网页

hexo s  #重启预览

hexo d  #向远端部署


##注意重启预览往往对应localhost:4000本地网页，只有向远端部署后，才能利用远端网址打开博客。
##主题更新可能有延迟，可稍作等待。或多次刷新Github仓库，再利用远端网址打开博客。
